Magasinguide

Sesong styrer hele reisen

For Phuket, Krabi, Khao Lak og mange øyer på Andaman-kysten er november til april vanligvis enklest. For Koh Samui, Koh Phangan og Koh Tao kan sommermånedene ofte fungere bedre enn vestkysten.

Bangkok kan besøkes hele året, men varme, regn og luftkvalitet kan påvirke tempoet. Planlegg stranddelen først, og legg byen inn der fly og rytme passer.

Samme struktur som Phuket-siden: rask konklusjon, valg som betyr noe, og lenker videre til riktig guide.

Verdt å vite

  • November-april er tryggest for Phuket, Krabi og Khao Lak.
  • Juni-august peker ofte mot Koh Samui, Koh Phangan og Koh Tao.
  • Jul, nyttår og vinterferie bør bestilles tidlig.
